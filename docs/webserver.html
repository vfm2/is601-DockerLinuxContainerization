Setup a Web Server with Docker

A tutorial on how to setup a web server on your local computer using docker

Step One: Build the Docker File

docker pull ubuntu

Creates the image from the Ubuntu base image

docker run -it ubuntu:latest /bin/bash

starts and run a new docker container with image "ubuntu:latest"

***Now that you are in the image, lets install and update the following packages: apt-get update

updates all the packages on the Ubuntu system

apt-get install apache2

installs apache2 onto the image

apt-get install vim

install vim onto the image

apt-get clean

cleans up any unnecessary files from the system

***Now that our image is ready, lets exit the container using "exit" command and commit your container

Step Two: Commit the Docker Image

docker ps -a

displays all containers

docker commit <containerID> <nameOfNewDockerImage>

Commits your docker image created into a new docker image

Step Three: Run New Docker Image on Local Web Server

docker run -it -p 127.0.0.1:80:80 -v C:\File_Path\getting-started:/var/www/html ubuntu-my_apache2 /bin/bash

it will login to an interactive shell, -p will expose the port, and -v will bind the host directory.

You can now open up your browser and input "127.0.0.1". You will be able to view your local docker image!
