Connect Docker with Pycharm

A tutorial about how to connect Docker with Pycharm to write python code

Add Docker and Python Files

  1. Open a new python project in PyCharm
  2. Create new file named "Dockerfile" and input the following code to get started:
    3.           
            FROM python:3

            ADD my_script.py /

            RUN pip install pystrich

            CMD [ "python", "./my_script.py" ]
  3. Create new python file named "my_script.py" and input the following code:
    4.           
            # Sample taken from pyStrich GitHub repository
            # https://github.com/mmulqueen/pyStrich
            from pystrich.datamatrix import DataMatrixEncoder

            encoder = DataMatrixEncoder('This is a DataMatrix.')
            encoder.save('./datamatrix_test.png')
            print(encoder.get_ascii())

Connect to Docker

  1. Go to "File" and select "Settings"
    1. A new window will pop open with PyCharm Setting
  2. Open the "Build, Execution, Deployment" tab, select Docker
    1. Click on the plus on the top left corner
    2. Name will be defaulted to "Docker"
    3. Select "TCP socket"
    4. Type "tcp://localhost:2375" in Engine API URL box
    5. Wait for bottom box to input "Connection successful"


  3. Click on "Docker Registration" under "Docker" on left side
    1. Click on the plus on the top left corner
    2. Input your docker username, password, and email"
    3. Wait for bottom section to input "Connection successful"


  4. Click "Apply" and "OK" on bottom right of screen
  5. Click on dropdown on top right of screen, then "Edit Configurations"
  6. Find "Add" plus sign on top left of window
  7. Select "Docker" dropdown, then "Dockerfile"
    1. Input Name, Server (default Docker), Dockerfile (select dockerfile previously created) and image tag name
    2. confirm "Run built image" is checked
    3. Click "Apply" and "OK"


  8. Next to dropdown on top right, confirm docker name is correct and click the play button to build the image

After image is built, time to create the interpreter

  1. On the bottom right, select "Add Interpreter"
  2. Select "Docker"
  3. Confirm Image Name is correct then click "OK"

Your PyCharm project is connected to Docker!
