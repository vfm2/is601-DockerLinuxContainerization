Linux and Docker Commands

Basic Linux commands and Login to your Docker Container

Linux Commands

pwd

Displays the current directory path

cd ../folder/subfolder

CD directs you to a new directory. This command above will move one directory back, and move into path folder/subfolder

mkdir /tmp/newFolder

This makes a directory in the path given. The command above will create "newFolder" in "tmp" folder

ls

Lists all folders and files

cat tutorial.txt

cat will show the content of the file "tutorial.txt"

mv moveFile.txt toThisFoler

mv command will move "moveFile.txt" to folder "toThisFolder". mv command will allow you to move more than one file.



Login to your Docker Container

  • docker ps
    • lists out all existing docker images and names on your machine


  • docker exec -it <container> /bin/bash
    • to get a bash shell into an existing container


  • docker run -it <image name> <command>
    • To run a container that is not defined by a docker-compose.yml file
    • Use docker run to start a new container with a given image
